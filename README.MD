# React Learning Journey

This repository contains all the projects and exercises I built while learning **React.js** from **Hitesh Choudhary's React Course**. Each folder focuses on a different React concept, starting from the basics and gradually moving towards more advanced topics like Context API, Redux Toolkit, Routing, and Backend Integration.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/GaneshNaik2006/React.git
```

Move into the project folder:

```bash
cd React
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
